Notatki z nauki o CSS

* skłądnia CSS to są reguły = {}
* selektor = za pomocą selektora wybieramy co chcemy osterować, a w nawiasie wpisujemy właściwość czyli to
co chcemy zrobić z selektorem np. kolor tła, kolor sekcji, przesunięcie tekstu itp.
* najlepiej stylować w osobnym pliku i zalinkować ten plik do htmlu.
* istnieje sposób stylowania liniowego czyli wpisujemy w dany element itp style i wpisujemy porządaną
właściwość, ale jest sposób chaotyczny.
* fajnym sposobem jest stylowanie poprzez classy, czyli nadajemy nazwę class danemu elemtowi i wpisujemy w pliku style porządaną właściwość
* kolejny sposób jest poprzez nadanie id i # w pliku ze stylami
* fajna strona do generowania kolorów
